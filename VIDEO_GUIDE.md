# Pre-Recorded Demo Video — What to Record

**Suggested length:** 2–3 minutes  
**Goal:** Judges can watch without running the app.

---

## Recording Setup

- **Tool:** OBS, Loom, or screen + webcam recording
- **Resolution:** 1080p
- **Include:** Dashboard in full view, optionally your face (picture-in-picture)

---

## Video Structure (Shot List)

### Scene 1: Intro (15 sec)
**Show:** Dashboard with live feed, traffic lights, status cards.

**Script:**  
"S.U.T.R.A. — Smart Urban Traffic Response Assistant. An AI-powered traffic management system for Indian intersections."

---

### Scene 2: Normal Operation (15 sec)
**Show:** Traffic lights cycling, countdown timer, Lane A / B counts.

**Script:**  
"The system runs adaptive traffic control with normal cycling and vehicle counting."

---

### Scene 3: Signal for Help (30 sec)
**Show:** You performing the gesture (palm → thumb tucked → fist 3×). Safety card turning red, log message, alert sound.

**Script:**  
"For vulnerable users, we support the Signal for Help gesture. Palm open, thumb tucked, fist three times. The system detects it and dispatches police."

---

### Scene 4: Green Corridor (30 sec)
**Show:** Truck or siren trigger. Green Corridor ACTIVE, EVP override, lane forced green.

**Script:**  
"When an ambulance approaches, we create a green corridor. Signals turn green so the ambulance passes without stopping — like India’s organ transport corridors."

---

### Scene 5: Festival Mode (15 sec)
**Show:** Toggle Festival Mode ON. V2I status changes.

**Script:**  
"For festivals and processions, Festival Mode extends pedestrian crossing time."

---

### Scene 6: Demo Mode (20 sec)
**Show:** Type `/demo` in the AI Command Panel. Simulated phases cycle: Ambulance, SOS, Accident, Animal.

**Script:**  
"Without hardware, Demo Mode cycles through all events. Judges can try this with ?demo=1 in the URL."

---

### Scene 7: Recording & Event Log (15 sec)
**Show:** Event log with filters (All, SOS, Accident), refresh button. Mention recordings in `recordings/`.

**Script:**  
"Incidents are logged. When SOS, ambulance, or accident is detected, we record a short video clip for evidence."

---

### Scene 8: Outro (10 sec)
**Show:** Dashboard overview or your face.

**Script:**  
"S.U.T.R.A. — built for India’s traffic. Thank you."

---

## Tips

1. **Prepare in advance:** Run the app, trigger each feature once, then record in one take.
2. **Keep it tight:** 2 minutes is enough; extra length only if you add more features.
3. **Add captions:** Help viewers follow even without sound.
4. **Upload to:** YouTube (unlisted) or Loom; add the link to your README or submission.
